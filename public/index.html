Mis Proyectos

Sitio Web de Comercio Electrónico

Desarrollo de un sitio web responsivo para una tienda en línea que incluye un catálogo de productos, carrito de compras y un sistema de pagos seguro.

Tecnologías utilizadas: HTML, CSS, JavaScript, React, Node.js, MongoDB.

Características destacadas:

  • Diseño intuitivo y atractivo.
  • Gestión de inventario en tiempo real.
  • Integración con pasarelas de pago como PayPal y Stripe.

Aplicación de Gestión de Tareas

Aplicación móvil para la gestión de tareas, que permite a los usuarios crear, editar y eliminar tareas, establecer recordatorios y sincronizar datos entre dispositivos.

Tecnologías utilizadas: React Native, Redux, Firebase.

Características destacadas:

  • Notificaciones push para recordatorios.
  • Sincronización en tiempo real entre múltiples dispositivos.
  • Interfaz de usuario limpia y minimalista.

Blog Personal con CMS

Creación de un blog personal con un sistema de gestión de contenido (CMS) personalizado para facilitar la publicación y edición de artículos.

Tecnologías utilizadas: WordPress, PHP, MySQL, CSS, JavaScript.

Características destacadas:

  • Editor de texto enriquecido para la creación de contenido.
  • Sistema de comentarios para interacción con los lectores.
  • SEO optimizado para mejorar la visibilidad en motores de búsqueda.

Plataforma de Aprendizaje en Línea

Desarrollo de una plataforma de aprendizaje en línea que ofrece cursos en diversos temas, con video lecciones, quizzes y seguimiento de progreso.

Tecnologías utilizadas: Django, Python, PostgreSQL, HTML, CSS, JavaScript.

Características destacadas:

  • Registro y gestión de usuarios.
  • Sistema de evaluación con quizzes interactivos.
  • Panel de administración para la gestión de cursos y contenido.

Aplicación de Clima

Aplicación web que proporciona información meteorológica en tiempo real para cualquier ubicación, con gráficos de pronóstico y alertas de clima severo.

Tecnologías utilizadas: Vue.js, Axios, API de OpenWeatherMap.

Características destacadas:

  • Búsqueda por ciudad y ubicación geográfica.
  • Visualización de datos meteorológicos actuales y pronósticos a 7 días.
  • Alertas personalizadas para condiciones climáticas severas.

Juego Web Interactivo

Desarrollo de un juego web interactivo, tipo puzzle, que desafía a los usuarios a resolver acertijos para avanzar a través de diferentes niveles.

Tecnologías utilizadas: HTML5, CSS3, JavaScript, Phaser.js.

Características destacadas:

  • Gráficos y animaciones atractivas.
  • Niveles progresivos con dificultad creciente.
  • Tabla de puntuaciones para competir con otros jugadores.
